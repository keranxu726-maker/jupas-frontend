## 📖 快速体验流程

### 学生端体验（5分钟）

1. **登录** - 使用 student1 / 123456
2. **输入成绩**
   - 填写3门必选科目（Chinese, English, Math）
   - 添加2门以上选修科目（如Physics, Chemistry）
   - 为每门科目选择成绩等级（建议选5或5*）
3. **查看推荐**
   - 点击"计算专业推荐"
   - 查看总分和推荐专业列表
   - 注意成功率颜色（绿🟢高/橙🟠中/红🔴低）
4. **收藏专业**
   - 点击星标⭐收藏喜欢的专业
   - 访问"我的收藏"查看
5. **修改密码**
   - 进入"系统设置"
   - 测试密码重置功能

### 管理端体验（3分钟）

1. **登录** - 使用 admin / admin123
2. **账号管理**
   - 切换"学生账号"/"管理员账号"标签
   - 尝试新增/编辑/删除账号
   - 为学生账号设置权益点数
3. **规则管理**
   - 上传一个JSON文件
   - 预览规则内容
4. **专业数据**
   - 搜索专业
   - 查看数据列表

## 💡 使用技巧

### 学生端
- 每次计算消耗1个权益点数
- 可以反复调整成绩重新计算
- 收藏的专业会保存在浏览器中
- 筛选功能支持模糊搜索

### 管理端
- 管理员可以互相维护
- 学生权益点数可随时调整
- JSON规则文件支持多个上传
- 专业数据支持实时搜索
