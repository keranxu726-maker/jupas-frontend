# JUPAS 成绩计算与专业推荐系统

现代专业的蓝色主题前端应用 🎨

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问：http://localhost:3000
